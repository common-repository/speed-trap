Simple lightweight Plugin to track all WordPress pages and load times, including the admin area.

== Description ==

Tracks and logs all the pages accessed on the front end and admin section of your WordPress site. Each page accessed, load time, date, and user (if logged in) is stored. The access logs can be viewed by administrators via the Plugin settings page.

== Installation ==

Instructions for installing the Speed Trap Plugin.

1. In the WordPress admin go to Plugins => Add New.
2. Enter 'Speed Trap' (without quotes) in the textbox and click the 'Search Plugins' button.
3. In the list of relevant Plugins click the 'Install' link on the right hand side of the page.
4. Click the 'Install Now' button on the popup page.
5. Click 'Activate Plugin' to finish installation.
6. Refresh any of your front end site pages that may be open.
7. That's it!
